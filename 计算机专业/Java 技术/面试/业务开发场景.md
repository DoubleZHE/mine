## 扫码登录的实现方式

扫码登录的**本质**：通过已登录的 APP 应用去扫描未登录的 WEB 端程序的二维码，通过某种机制触发登录凭证的写入，从而实现登录。

`核心设计：特定的二维码去连接 APP 端和网页端。`
